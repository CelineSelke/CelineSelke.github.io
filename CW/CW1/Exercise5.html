Exercise 5

Easy options for maintaining quality in your website's design include making sensible color choices and emphasizing usability by making your layout easy to understand. In addition, you should account for different browsers and platforms. For example, many users browse the Internet using their smartphones, so the website must either be formatted with that in mind or provide an equivalent version with a layout more suited for that platform.
Differing stylings between pages creates a sense a disjointedness in the website when viewed as a whole. It is very unlikely that users will only ever use a single page of your website, so consistency is key in creating a cohesive experience. In addition, it is unlikely in this particular case that each individual teacher will have adequate knowledge of web design, so pushing the expectations of quality design onto each individual will lead to gaps in quality between pages as well.
  • Common HTML Tags
    • <p>: Standing for paragraph, this tag is common for basic text
    • <a>: Refers to hyperlinks, which are required for navigating between pages
    • <div>: Standing for division, this tag creates a segment of the page which can itself hold elements, which makes positioning and styling groups of elements much easier
    • <img>: Refers to images, which are helpful in bringing pages to life and can be used to hold links that aren't connected to any text
    • <h1>,<h2>,<h3>, etc.: Headings which are helpful for establishing sections of a page for the viewer
  • Common CSS Attributes
    • text-align: positions text with options for centering or aligning text to a specific end of the container
    • font-size: changes the size of text
    • font-family: changes the font used in a section of text
    • color: changes the color of text
    • margin(-[position]): controls the space surrounding the element in question
    • padding(-[position]): controls the space around an element within a container
